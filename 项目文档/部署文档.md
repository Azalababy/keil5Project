# 校园外卖系统 - 项目部署文档

## 1. 项目概述

校园外卖系统是一个面向校园场景的在线外卖订餐平台，支持用户、商家、骑手三种角色，提供完整的订餐、配送、评价等业务流程。

## 2. 系统要求

### 2.1 运行环境要求

- **JDK版本**: JDK 11 或更高版本
- **Node.js版本**: Node.js 16.x 或更高版本
- **数据库**: MySQL 8.0 或更高版本
- **Maven版本**: Maven 3.6 或更高版本
- **操作系统**: Windows 10/11 或 Linux (Ubuntu/CentOS)

### 2.2 开发工具推荐

- **IDE**: 
  - 后端: IntelliJ IDEA 或 Eclipse
  - 前端: VS Code
- **数据库管理工具**: MySQL Workbench 或 Navicat
- **版本控制**: Git

## 3. 数据库部署

### 3.1 安装MySQL

#### Windows系统

1. 下载MySQL安装包: https://dev.mysql.com/downloads/mysql/
2. 选择MySQL Installer进行安装
3. 配置root用户密码（建议设置为 `123456`）
4. 完成安装

#### Ubuntu系统

```bash
sudo apt update
sudo apt install mysql-server
sudo systemctl start mysql
sudo systemctl enable mysql
```

#### CentOS系统

```bash
sudo yum install mysql-server
sudo systemctl start mysqld
sudo systemctl enable mysqld
```

### 3.2 创建数据库

1. 登录MySQL

```bash
mysql -u root -p
```

2. 创建数据库

```sql
CREATE DATABASE meituan CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE meituan;
```

3. 导入数据库备份（可选）

```bash
mysql -u root -p meituan < backup/meituan_backup_xxxxxx.sql
```

### 3.3 数据库配置

配置文件位置: `src/main/resources/application.properties`

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/meituan?useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true
spring.datasource.username=root
spring.datasource.password=123456
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver
```

> **注意**: 如果修改了数据库用户名或密码，请同时修改上述配置文件中的对应值。

## 4. 后端部署

### 4.1 环境准备

#### 安装JDK

1. 下载JDK 11+: https://www.oracle.com/java/technologies/downloads/
2. 安装并配置环境变量
3. 验证安装

```bash
java -version
```

#### 安装Maven

1. 下载Maven: https://maven.apache.org/download.cgi
2. 解压到指定目录（如: `D:\apache-maven`）
3. 配置环境变量
4. 验证安装

```bash
mvn -version
```

### 4.2 项目配置

1. 修改数据库连接配置

打开 `src/main/resources/application.properties` 文件，修改数据库连接信息：

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/meituan
spring.datasource.username=root
spring.datasource.password=123456
```

2. （可选）修改服务器端口

默认端口为8080，如需修改请在配置文件中添加：

```properties
server.port=8080
```

### 4.3 编译和运行

#### 开发环境运行

```bash
cd c:\Users\Administrator\Desktop\waimai
mvn spring-boot:run
```

或使用IDE直接运行主类: `CampusFoodDeliveryApplication.java`

#### 打包部署

```bash
mvn clean package -DskipTests
```

打包完成后，在 `target` 目录下会生成 `campus-food-delivery-0.0.1.jar` 文件。

#### 运行JAR包

```bash
java -jar target/campus-food-delivery-0.0.1.jar
```

### 4.4 验证后端服务

启动成功后，访问 http://localhost:8080/ 或 http://localhost:8080/api 测试服务是否正常运行。

## 5. 前端部署

### 5.1 环境准备

#### 安装Node.js

1. 下载Node.js: https://nodejs.org/
2. 安装LTS版本
3. 验证安装

```bash
node -v
npm -v
```

### 5.2 项目配置

1. 安装依赖

```bash
cd frontend
npm install
```

2. 配置API地址

前端默认连接后端地址: `http://localhost:8080`

如需修改，请编辑 `vite.config.ts` 文件中的代理配置。

### 5.3 开发环境运行

```bash
cd frontend
npm run dev
```

前端启动后会自动打开浏览器，默认访问地址: http://localhost:3000

> **注意**: 如果端口3000被占用，系统会自动选择下一个可用端口（3001、3002等）。

### 5.4 生产环境构建

```bash
cd frontend
npm run build
```

构建完成后，静态文件会生成在 `dist` 目录中。

### 5.5 部署静态文件

#### 使用Nginx部署

1. 安装Nginx

**Windows**: 下载nginx for Windows
**Ubuntu**: `sudo apt install nginx`
**CentOS**: `sudo yum install nginx`

2. 配置Nginx

编辑nginx配置文件，添加以下内容：

```nginx
server {
    listen 80;
    server_name your_domain.com;
    
    location / {
        root /path/to/campus-food-delivery/frontend/dist;
        try_files $uri $uri/ /index.html;
    }
    
    location /api {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

3. 启动Nginx

```bash
nginx -t  # 测试配置
nginx     # 启动服务
```

## 7. 网络访问配置

### 7.1 防火墙设置

#### Windows防火墙

允许以下端口通过防火墙：
- 8080 (后端API)
- 3000/3001/3002 (前端服务)

```powershell
netsh advfirewall firewall add rule name="Backend API" dir=in action=allow protocol=tcp localport=8080
netsh advfirewall firewall add rule name="Frontend" dir=in action=allow protocol=tcp localport=3000
```

#### Ubuntu/CentOS防火墙

```bash
# Ubuntu (UFW)
sudo ufw allow 8080
sudo ufw allow 3000
sudo ufw reload

# CentOS (Firewalld)
sudo firewall-cmd --permanent --add-port=8080/tcp
sudo firewall-cmd --permanent --add-port=3000/tcp
sudo firewall-cmd --reload
```

### 7.2 获取本机IP地址

#### Windows

```powershell
ipconfig | Select-String "IPv4"
```

#### Linux/Mac

```bash
hostname -I
```

### 7.3 移动端访问

在同一局域网下，使用手机访问：
```
http://<电脑IP地址>:3000
```

示例: `http://192.168.1.100:3000`

## 8. 数据库备份与恢复

### 8.1 手动备份

```bash
mysqldump -u root -p meituan > backup_$(date +%Y%m%d).sql
```

### 8.2 恢复数据

```bash
mysql -u root -p meituan < backup_20240625.sql
```

### 8.3 自动备份（Linux Cron）

创建备份脚本 `/backup/mysql_backup.sh`:

```bash
#!/bin/bash
DATE=$(date +%Y%m%d_%H%M%S)
mysqldump -u root -p123456 meituan > /backup/meituan_$DATE.sql
find /backup -name "meituan_*.sql" -mtime +7 -delete
```

添加定时任务:

```bash
crontab -e
# 添加以下行，每天凌晨2点执行备份
0 2 * * * /backup/mysql_backup.sh
```

## 9. 常见问题

### 9.1 后端启动失败

**问题**: 数据库连接失败

**解决方案**:
1. 确认MySQL服务已启动
2. 检查数据库用户名和密码
3. 确认数据库 `meituan` 已创建

### 9.2 前端无法访问后端API

**问题**: CORS跨域错误

**解决方案**:
1. 确认后端服务已启动
2. 检查 `vite.config.ts` 中的代理配置
3. 确认前端API请求地址正确

### 9.3 端口被占用

**问题**: 端口已被其他程序占用

**解决方案**:
1. 查找占用端口的进程: `netstat -ano | findstr :8080`
2. 结束进程或更换端口
3. 修改 `application.properties` 中的端口配置

### 9.4 移动端无法访问

**问题**: 防火墙阻止

**解决方案**:
1. 关闭防火墙进行测试
2. 或添加防火墙规则允许相应端口
3. 确保手机和电脑在同一局域网
