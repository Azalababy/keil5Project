# 校园外卖系统 - 技术架构文档

## 1. 系统架构概述

校园外卖系统采用前后端分离的B/S架构，后端使用Spring Boot框架提供RESTful API，前端使用Vue3框架构建单页面应用，数据库采用MySQL进行数据持久化存储。


## 6. 核心业务逻辑

### 6.1 订单创建流程

```java
public Order createOrder(OrderRequest request) {
    // 1. 验证用户信息
    User user = userRepository.findById(request.getUserId())
        .orElseThrow(() -> new RuntimeException("用户不存在"));
    
    // 2. 验证商家信息
    Shop shop = shopRepository.findById(request.getShopId())
        .orElseThrow(() -> new RuntimeException("商家不存在"));
    
    // 3. 计算订单金额
    BigDecimal totalPrice = calculateTotalPrice(request.getItems());
    
    // 4. 应用优惠券
    BigDecimal discount = BigDecimal.ZERO;
    if (request.getCouponId() != null) {
        discount = applyCoupon(request.getUserId(), request.getCouponId(), totalPrice);
    }
    
    // 5. 创建订单
    Order order = new Order();
    order.setUserId(request.getUserId());
    order.setShopId(request.getShopId());
    order.setAddress(request.getAddress());
    order.setPhone(request.getPhone());
    order.setTotalPrice(totalPrice);
    order.setActualPrice(totalPrice.subtract(discount));
    order.setCouponId(request.getCouponId());
    order.setStatus(Order.Status.pending);
    
    order = orderRepository.save(order);
    
    // 6. 创建订单项
    for (OrderItemRequest item : request.getItems()) {
        OrderItem orderItem = new OrderItem();
        orderItem.setOrderId(order.getId());
        orderItem.setDishId(item.getDishId());
        orderItem.setDishName(item.getDishName());
        orderItem.setPrice(item.getPrice());
        orderItem.setQuantity(item.getQuantity());
        orderItemRepository.save(orderItem);
    }
    
    // 7. 更新优惠券状态
    if (request.getCouponId() != null) {
        userCouponRepository.updateStatus(request.getUserId(), 
                                          request.getCouponId(), 
                                          order.getId());
    }
    
    return order;
}
```

### 6.2 优惠券领取逻辑

```java
public Map<String, Object> grabCoupon(Long userId, Long couponId) {
    // 1. 验证优惠券存在
    Coupon coupon = couponRepository.findById(couponId)
        .orElseThrow(() -> new RuntimeException("优惠券不存在"));
    
    // 2. 检查优惠券状态
    if (coupon.getStatus() != Coupon.Status.active) {
        throw new RuntimeException("优惠券已下架");
    }
    
    // 3. 检查过期时间
    if (coupon.getExpireDate().isBefore(LocalDate.now())) {
        throw new RuntimeException("优惠券已过期");
    }
    
    // 4. 检查库存
    if (coupon.getRemainingQuantity() <= 0) {
        throw new RuntimeException("优惠券已抢光");
    }
    
    // 5. 检查领取限制
    long count = userCouponRepository.countByUserIdAndCouponId(userId, couponId);
    if (count >= 1) {
        throw new RuntimeException("您已经领取过该优惠券");
    }
    
    // 6. 扣减库存
    coupon.setRemainingQuantity(coupon.getRemainingQuantity() - 1);
    couponRepository.save(coupon);
    
    // 7. 创建用户优惠券记录
    UserCoupon userCoupon = new UserCoupon();
    userCoupon.setUserId(userId);
    userCoupon.setCouponId(couponId);
    userCoupon.setStatus(UserCoupon.Status.unused);
    userCouponRepository.save(userCoupon);
    
    return Map.of("success", true, "message", "领取成功");
}
```
